### Mockserver
==========
set up your api server using local config file.

[![Build Status](https://travis-ci.org/whwei/mockserver.svg?branch=master)](https://travis-ci.org/whwei/mockserver)


### Usage
=========
```
mockserver -d path-to-config-file.json
```

### Options
=========
-   `-d`: config file path
-   `-p`: port
-   `-h`: host
